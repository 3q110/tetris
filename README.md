# 俄罗斯方块 Tetris

> [🎮 在线试玩](https://3q110.github.io/tetris/)

经典俄罗斯方块游戏，包含终端版（Python curses）和 HTML 浏览器版（Canvas）。

## 玩法

| 按键 | 功能 |
|------|------|
| `空格` | 开始 / 暂停 / 重新开始 |
| `↑` | 旋转方块 |
| `←` `→` | 左右移动 |
| `↓` | 加速下落 |

## 功能

- 七种经典方块造型，彩色显示
- 方块碰撞边界检测
- 自动下落 + 手动操控
- 满行消除得分
- 等级递增，难度自动提速
- 右侧下一方块预览
- 游戏暂停 / 继续 / 结束重置

## 文件说明

| 文件 | 说明 |
|------|------|
| `俄罗斯方块.md` | 设计文档 |
| `俄罗斯方块.py` | 终端版（Python curses） |
| `俄罗斯方块.html` | HTML 浏览器版（Canvas） |
| `index.html` | 在线试玩入口（同 HTML 版） |

## 运行方式

### HTML 浏览器版（推荐）

直接用浏览器打开 `俄罗斯方块.html` 即可游玩。

### 终端版

```bash
python3 俄罗斯方块.py
```

需要 Python 3 环境，使用 `curses` 库（Linux / macOS 内置，Windows 需安装 `windows-curses`）。
