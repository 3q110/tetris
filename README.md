# 俄罗斯方块 Tetris

> [🎮 在线试玩](https://3q110.github.io/tetris/)

经典俄罗斯方块游戏，包含终端版（Python curses）和 HTML 浏览器版（Canvas）。

## 玩法

| 按键 | 功能 |
|------|------|
| `空格` | 开始 / 暂停 / 重新开始 |
| `↑` | 旋转方块 |
| `←` `→` | 左右移动 |
| `↓` | 加速下落 |
| `W` | 一键落底 |

## 功能

- 七种经典方块造型，彩色显示
- 方块碰撞边界检测
- 自动下落 + 手动操控
- 满行消除得分
- 等级递增，难度自动提速
- 右侧下一方块预览
- 游戏暂停 / 继续 / 结束重置
- 最高分记录（localStorage 持久化）
- 幽灵方块（落点预览）
- DAS 长按加速移动

## ✨ 终极版新增

- **音效系统** — 移动、旋转、落底、消除、升级、游戏结束均有独立音效（Web Audio API，无需外部文件）
- **消除爆炸效果** — 消除行时产生彩色粒子爆炸 + 屏幕震动 + 闪光
- **落底特效** — 一键落底时产生地面冲击粒子
- **UI 升级** — 星空背景、方块发光、分数弹出动画、面板悬停效果
- **音效开关** — 右上角 🔊 按钮可切换音效

## 运行方式

### HTML 浏览器版（推荐）

直接用浏览器打开 `俄罗斯方块.html` 即可游玩。

### 终端版

```bash
python3 俄罗斯方块.py
```

需要 Python 3 环境，使用 `curses` 库（Linux / macOS 内置，Windows 需安装 `windows-curses`）。
